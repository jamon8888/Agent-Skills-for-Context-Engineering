# Instructions d'Analyse : ADN Vocal

### Étape 1 : Collecte

- **Smart Scan** : Recherchez dans `${CLAUDE_PLUGIN_ROOT}/data/0-Inbox` ou un dossier spécifié par l'utilisateur.
- **Filtrage** : Privilégiez `.md`, `.txt`, `.docx`.

### Étape 2 : Analyse

- **Marqueurs** : Énergie, empathie, formalité.
- **Motifs** : Longueur des phrases, usage des emojis, ponctuation.
- **Vocabulaire** : Mots signatures, transitions préférées.
- **Ce qu'il faut éviter** : Liste noire de termes ou de tons.

### Étape 3 : Validation

Générez 3 phrases dans la voix capturée pour approbation.
