# RFP Red Flags (The Deal Killers)

If you see these terms, the probability of winning drops by 50%+.

## 1. Contractual Nightmares

- **"Unlimited Liability"**: Never sign this.
- **"Source Code Escrow"**: They want your code if you go bust.
- **"Liquidated Damages"**: Getting fined for delays.
- **"Most Favored Customer"**: You can never sell cheaper to anyone else.

## 2. Technical Impossibilities

- **"On-Premise Only"**: If you are SaaS, this is a NO-GO.
- **"Custom Development"**: Check if paid. If free, NO-GO.
- **"Source Code Access"**: Taking over your IP.

## 3. Process Flags

- **"Due in 48 hours"**: The winner is already chosen; they just need 3 bids for compliance.
- **"Blind Submission"**: No Q&A allowed.
- **"Generic Excel Sheet"**: 500 rows of generic demands (feature shopping).

## Bilingual Keywords (French)

- **"Pénalités de retard"** (Liquidated Damages)
- **"Responsabilité illimitée"** (Unlimited Liability)
- **"Hébergement sur site obligatoires"** (On-Premise)
